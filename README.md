    # ClubHub

Sistema completo de gestão para clubes esportivos com gestão de atletas, eventos, presenças, escalas e finanças.

## Tecnologias

- React + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Supabase (Auth + Database)
- React Query

## Como executar localmente

Pré-requisitos: Node.js & npm instalados - [instalar com nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Clone o repositório
git clone <YOUR_GIT_URL>

# Navegue até o diretório do projeto
cd clubhub

# Instale as dependências
npm i

# Inicie o servidor de desenvolvimento
npm run dev
```
- Edit files directly within the Codespace and commit and push your changes once you're done.
